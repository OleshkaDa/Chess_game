# Chess Game

## Требования

Для запуска проекта необходимо установить:

* Visual Studio 2022
* .NET 8 SDK
* Git (необязательно, если скачивать проект архивом)

---

## Загрузка проекта

### Вариант 1. Через GitHub

1. Откройте страницу проекта на GitHub.
2. Нажмите зеленую кнопку **Code**.
3. Выберите **Download ZIP**.
4. Распакуйте архив в любую папку на компьютере.

### Вариант 2. Через Git

Откройте терминал и выполните команду:

git clone Ссылка на репрезтнторий

После загрузки откройте папку проекта.

---

## Открытие проекта

1. Запустите Visual Studio 2022
2. Нажмите Open a project or solution
3. Перейдите в папку проекта.
4. Откройте файл решения (*.sln)
5. Дождитесь восстановления пакетов NuGet.

---

## Запуск игры

1. Нажмите кнопку **Start** (зелёный треугольник) в верхней части Visual Studio.
2. Дождитесь сборки проекта.
3. После успешной сборки откроется главное меню игры.

---

## Начало новой игры

1. Нажмите кнопку **Новая игра**.
2. Откроется игровое окно с шахматной доской.
3. Для хода нажмите на фигуру, затем на клетку назначения.

---

## Сохранение игры

1. Во время игры нажмите кнопку **Сохранить**.
2. Сохранение будет создано в папке:

Документы/ChessSaves

3. Формат сохранения зависит от выбранного формата:

   * JSON
   * XML

---

## Загрузка сохранения

1. В главном меню выберите формат сохранения.
2. Нажмите **Продолжить игру**.
3. Выберите нужный файл сохранения.
4. После загрузки игра продолжится с сохранённого состояния.

---

## Дополнительные возможности

* Отмена последнего хода.
* Проверка шаха и мата.
* Определение победителя.
* Сохранение и загрузка партии.
* Поддержка двух форматов хранения данных (JSON и XML).

---

## Используемые технологии

* C#
* WPF
* .NET 8
* JSON Serialization
* XML Serialization
