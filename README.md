# Chess

## Описание проекта

Chess — настольная игра в шахматы для двух игроков, разработанная на C# с использованием WPF.

Проект поддерживает:

* создание новой партии;
* продолжение сохранённой партии;
* сохранение и загрузку игры;
* выбор формата сохранения (JSON или XML);
* определение шаха, мата и пата;
* отмену хода;
* подсветку доступных ходов.

---

## Главное меню

После запуска приложения открывается главное меню.

### Кнопка «Новая игра»

Создаёт новую шахматную партию с классической расстановкой фигур.

После нажатия открывается окно игры.

---

### Кнопка «Продолжить игру»

Загружает ранее сохранённую партию.

Кнопка активна только в случае, если выбран корректный файл сохранения.

Если выбран повреждённый файл или файл неверного формата, выводится сообщение:

```
Файл поврежден
```

и кнопка становится недоступной.

---

### Поле выбора папки сохранений

Позволяет выбрать папку, в которой будут храниться файлы сохранений.

Для выбора папки используется кнопка «...».

---

### Список выбора формата

Позволяет выбрать формат сохранения партии:

* JSON
* XML

Выбранный формат используется для сохранения текущей партии.

---

## Игровое окно

После создания новой партии или загрузки сохранения открывается игровое окно.

---

### Игровое поле

Игровое поле состоит из 64 клеток.

Для выполнения хода необходимо:

1. Нажать на свою фигуру.
2. Выбрать клетку для перемещения.

После успешного хода очередь автоматически переходит другому игроку.

---

### Подсветка ходов

После выбора фигуры клетки, доступные для перемещения, подсвечиваются зелёным цветом.

Выбранная фигура подсвечивается золотым цветом.

---

### Отмена хода

Кнопка «Отмена хода» возвращает игру на один ход назад.

Кнопка работает только при наличии ходов в истории партии.

---

### Кнопка «Сохранить»

Сохраняет текущее состояние партии в выбранную папку.

Имя файла формируется автоматически и содержит дату и время сохранения.

Пример:

```
chess_save_2025-06-22_14-30-15.json
```

---

### Кнопка «Выход»

Закрывает окно партии.

---

## Автосохранение

При закрытии игрового окна текущее состояние партии автоматически сохраняется в последний используемый файл сохранения.

---

## Шах

Если король игрока находится под атакой, отображается сообщение о шахе в области информации о текущем ходе.

---

## Мат

Если игрок не может защитить своего короля от шаха, партия завершается.

На экран выводится сообщение о победе соответствующего цвета фигур.

---

## Пат

Партия завершается ничьей в следующих случаях:

* игрок не находится под шахом и не имеет допустимых ходов;
* выполнено 6 взаимно обратных действий подряд.

При наступлении пата выводится сообщение:

```
Ничья
```

---

## Завершение работы

Для завершения игры необходимо закрыть окно партии или нажать кнопку «Выход».

После закрытия окна пользователь возвращается в главное меню.
